The computation of Shapley Values¶

v() = 0\ v(A) = 20\ v(B) = 20\ v(C) = 60\ v(A,B) = 60\ v(A,C) = 70\ v(B,C) = 70\ v(A,B,C) = 100

$$\frac{1}{3!}\sum_{S\in\{B,C,\{B,C\},\{\}\}}S!(n-|S|-1)!(v(S\bigcup\{i\}-v(S)) = \frac{1}{6}(v(A,B)-v(B)+v(A,C)-v(C)+2*(v(A,B,C)-v(B,C))+2*(v(A)-v())) = \frac{1}{6}(40+10+60+40)=25$$

The Shapley value of A with this value function is 25
